CSS is the language we use to style a Web page.

    CSS stands for Cascading Style Sheets
    CSS describes how HTML elements are to be displayed on screen, paper, or in other media
    CSS saves a lot of work. It can control the layout of multiple web pages all at once
    External stylesheets are stored in CSS files 

Basic Syntax
    h1 {
        color : red
    }

The selector points to the HTML element you want to style.
The declaration block contains one or more declarations separated by semicolons.
Each declaration includes a CSS property name and a value, separated by a colon.
Multiple CSS declarations are separated with semicolons, and declaration blocks are surrounded by curly braces.

--> There are 3 ways to add css

1. Inline CSS

    <h1 style="color:red">My name is Rahul</h1>

2. Internal CSS
    Note: <style> Tag is always used in the <head> Tag
    <head>
        <style>
            h1{
                color:red
            }
        </style>
    </head>

3. External CSS
    Note: To link the external file of the css (example: style.css), we write the code in the
    head Tag

    <head>
        <link rel="stylesheet" href="style.css">
    </head>
    

---> Color property
    Used to set the color of foreground
    eg:
        color:red;
        color:pink;
    
---> Background color  
    Used to set the color of background
    eg:
        background-color:red;
        background-color:pink;

COLOR SYSTEMS

--->RGB
    color: rgb(255,0,0)
    color: rgb(0,255,0)

--->Hex (Hexadecimal)
    color:#ff0000
    color:#00ff00


-----Selectors-----

1. Universal Selector
    *{
        background:red;
    }

2. Element Selector
    h1 {
    color:blueviolet;
    text-align: center;
    }
    p {
        color: brown;
        text-align: center;
    }

3. Id Selector
    #headingtwo {
    color:rgb(150, 55, 70);
    text-align: center;
}

4. Class Selector

.center {
    text-align: center;
}

.color {
    text-align: center;
    color: cadetblue;
}
